# AI-Based Study Resource Allocator

## 📌 Overview
An advanced resource allocation system that helps allocate study resources efficiently using scheduling algorithms and database management.

---

## 🚀 Features

- GUI using Tkinter
- Database connectivity using SQLite
- Priority-based resource allocation
- Progress tracking system
- Resource usage analysis
- Student management system

---

## 🛠️ Technologies Used

- Python
- Tkinter
- SQLite
- Pandas

---

## ▶️ How to Run

```bash
pip install -r requirements.txt
python allocator.py
